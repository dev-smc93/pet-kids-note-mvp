## 1. 앱 소개

### 1-1. 반려동물 알림장이란?

반려동물 돌봄 시설(유치원, 호텔, 센터 등)과 보호자 간에 **알림장**을 주고받는 서비스입니다.

- **관리자**: 원(시설)을 운영하며, 반려동물에게 알림장을 작성·수정·삭제합니다.
- **보호자**: 반려동물을 등록하고, 원과 연결된 후 알림장을 열람·작성합니다.

### 1-2. 역할별 주요 기능

| 역할       | 주요 기능                                                                      |
| ---------- | ------------------------------------------------------------------------------ |
| **관리자** | 원 생성·관리, 반려동물 연결 승인, 알림장 작성(생활기록 포함)·수정·삭제, 재알림 |
| **보호자** | 반려동물 등록, 원 검색·연결 요청, 알림장 작성·열람, 댓글                       |

### 1-3. 사용 흐름 요약

```
[관리자] 원 생성 → [보호자] 반려동물 등록 → [보호자] 원 검색·연결 요청
→ [관리자] 연결 승인 → [관리자/보호자] 알림장 작성·열람·댓글
```

## 2. 공통: 로그인·회원가입·프로필

### 2-1. 회원가입

1. 앱 접속 후 **회원가입** 클릭
2. 이메일, 비밀번호 입력
3. 가입 완료 후 **프로필 설정** 화면으로 이동

### 2-2. 프로필 설정 (역할 선택)

1. **닉네임** 입력
2. **역할** 선택
   - **관리자**: 원(시설)을 운영하는 경우
   - **보호자**: 반려동물을 맡기는 경우
3. **관리자** 선택 시 추가 입력
   - 원 이름
   - 시/도, 시/군/구, 주소
   - → 원이 자동으로 생성됩니다
4. **저장** 클릭

### 2-3. 로그인

1. 이메일, 비밀번호 입력
2. **로그인** 클릭
3. 역할에 맞는 홈 화면 표시

### 2-4. 프로필 수정

1. 상단 **프로필** 아이콘 클릭
2. 닉네임, 역할, (관리자 시) 원 정보 수정
3. **저장** 클릭

## 3. 공통: 푸시 알림 설정

알림장이 등록되거나 댓글이 달리면 **푸시 알림**을 받을 수 있습니다.

### 3-1. 알림 받기 (구독)

1. **프로필** 페이지 진입
2. **푸시 알림** 섹션에서 **알림 받기** 클릭
3. 브라우저에서 **알림 권한 허용** 선택
4. "알림 구독이 완료되었습니다." 메시지 확인

### 3-2. 알림 해제

1. **프로필** 페이지 진입
2. **알림 해제** 클릭
3. "알림 구독이 해제되었습니다." 메시지 확인

### 3-3. 푸시 알림이 오는 경우

| 상황        | 수신자                        |
| ----------- | ----------------------------- |
| 알림장 등록 | 보호자 (관리자가 작성한 경우) |
| 댓글 등록   | 상대방 (관리자 또는 보호자)   |
| 재알림 버튼 | 미열람 보호자                 |

## 4. 관리자: 원 관리

### 4-1. 원 목록 보기

1. 홈에서 **원 관리** 또는 **원** 메뉴 클릭
2. 내가 운영하는 원 목록 표시

### 4-2. 원 추가 (새 원 만들기)

1. 원 목록에서 **+** 또는 **원 추가** 클릭
2. 다음 정보 입력
   - 원 이름
   - 시/도, 시/군/구, 주소
3. **저장** 클릭

### 4-3. 원 상세 (승인 대기·연결된 반려동물)

1. 원 목록에서 원 선택
2. **승인 대기** 탭: 연결 요청을 한 보호자 목록
3. **연결된 반려동물** 탭: 이미 연결된 반려동물 목록

### 4-4. 연결 요청 승인/거절

1. 원 상세 → **승인 대기** 탭
2. 요청 목록에서 **승인** 또는 **거절** 클릭
3. 승인 시 해당 반려동물이 **연결된 반려동물**에 추가됨

### 4-5. 원에서 알림장 작성 (바로가기)

1. 원 상세 화면에서 **알림장 작성** 버튼 클릭
2. 해당 원 기준으로 알림장 작성 화면으로 이동

---

## 5. 관리자: 알림장 작성·수정·삭제·재알림

### 5-1. 알림장 작성

1. **알림장** 메뉴 클릭
2. 우측 하단 **연필 아이콘(FAB)** 클릭
3. **원 선택** (연결된 반려동물이 있는 원만 표시)
4. **대상 반려동물** 선택 (드롭다운)
5. **내용** 입력 (필수, 최대 5,000자)
6. **(선택) 사진 첨부**
   - **+** 클릭 후 이미지 1~10장 선택
   - JPEG/PNG/WebP, 파일당 10MB 이하
7. **(선택) 생활기록**
   - 하단 **생활기록** 버튼 클릭
   - 오버레이 패널에서 입력 후 폼에 반영
8. **작성** 클릭
   - 알림장 목록으로 이동, 작성 완료
   - 보호자가 알림 구독 중이면 푸시 알림 발송

### 5-2. 알림장 수정

1. 알림장 상세 화면에서 **수정** 버튼 클릭
2. 내용·사진 수정
3. **저장** 클릭

### 5-3. 알림장 삭제

1. 알림장 상세 화면에서 **삭제** 버튼 클릭
2. 확인 다이얼로그에서 **삭제** 확인
3. 알림장 목록으로 이동, 해당 알림장 삭제됨

### 5-4. 재알림

보호자가 아직 읽지 않은 알림장이 있을 때 사용합니다.

1. 알림장 상세 화면에서 **재알림** 버튼 클릭
2. "재알림 요청이 완료되었습니다." 메시지 확인
3. 보호자가 알림 구독 중이면 푸시 알림 발송

> 이미 읽은 알림장에는 재알림 버튼이 표시되지 않습니다.

### 5-5. 알림장 필터 (원 선택, 내가 쓴 글만)

1. 알림장 목록에서 **필터** 버튼 클릭
2. **원 선택**: 체크박스로 여러 원 선택 (전체 선택/선택 해제 가능)
3. **내가 쓴 글만 보기**: 토글 ON 시 내가 작성한 알림장만 표시
4. **적용** 클릭

---

## 6. 보호자: 반려동물 등록

### 6-1. 반려동물 목록

1. 홈에서 **내 반려동물** 또는 **반려동물** 메뉴 클릭
2. 등록한 반려동물 목록 표시

### 6-2. 반려동물 등록

1. **+** 또는 **반려동물 추가** 클릭
2. 다음 정보 입력
   - 이름 (필수)
   - 품종
   - 사진 (선택)
   - 특이사항 (메모)
3. **저장** 클릭

### 6-3. 반려동물 수정

1. 반려동물 목록에서 해당 반려동물 선택
2. **수정** 버튼 클릭
3. 정보 수정 후 **저장** 클릭

---

## 7. 보호자: 원 검색 및 연결 요청

### 7-1. 원 검색

1. **원 검색** 또는 **원 찾기** 메뉴 클릭
2. **시/도** 선택 (예: 서울, 경기)
3. 해당 지역의 원 목록 표시

### 7-2. 연결 요청

1. 연결하고 싶은 원 선택
2. **연결 요청** 클릭
3. **대상 반려동물** 선택 (내 반려동물 중)
4. **요청** 클릭
5. 관리자 승인 대기 상태로 전환

### 7-3. 연결된 원이 없을 때

- "연결된 원이 없습니다." 메시지 표시
- **원 검색에서 연결 요청하기** 링크로 원 검색 화면 이동

## 8. 알림장 목록·상세·댓글

> 관리자와 보호자 모두 사용하는 공통 기능입니다.

### 8-1. 알림장 목록

1. **알림장** 메뉴 클릭
2. 반려동물별 알림장 카드 표시
3. **미열람** 알림장에는 "새 알림" 배지 표시
4. 알림장이 없으면 "아직 받은 알림장이 없습니다." 메시지

### 8-2. 알림장 상세 보기

1. 알림장 카드 클릭
2. **제목** → **생활기록** → **이미지** → **내용** 순서로 표시
3. 사진 2장 이상: 슬라이더로 넘기며 확인
4. 아래에 더 내용이 있으면 스크롤 (화살표 안내)
5. **미열람** 알림장은 열람 시 자동으로 읽음 처리
6. 목록으로 돌아가면 "새 알림" 배지 사라짐

### 8-3. 댓글 작성

1. 알림장 상세 하단 **댓글** 섹션
2. 댓글 입력 후 **전송** 버튼 클릭
3. 댓글 목록에 추가, 상대방 구독 시 푸시 알림 발송

### 8-4. 댓글 예약

1. 댓글 입력창 옆 **시계** 버튼 클릭
2. 예약 시간 선택 (내일 8:30, 10:00, 15:00, 최근 예약, 직접 입력)
3. 댓글 입력 후 **전송**
4. 예약 댓글로 등록 (점선 테두리, "예약" 배지)
5. 예약 시간이 되면 자동으로 일반 댓글로 전환

### 8-5. 댓글 액션 (내 댓글 vs 상대방 댓글)

| 구분            | 가능한 동작                     |
| --------------- | ------------------------------- |
| **내 댓글**     | 댓글 복사, 댓글 수정, 댓글 삭제 |
| **상대방 댓글** | 댓글 복사                       |

- **댓글 복사**: 클립보드에 복사 + 입력창에 붙여넣기
- **예약 댓글**: "예약 취소" 버튼으로 삭제 가능

### 8-6. 댓글 실시간 반영

- 다른 기기/탭에서 새 댓글이 작성되면 **새로고침 없이** 즉시 반영됩니다.
- 예약 댓글은 예약 시간이 되면 자동으로 일반 댓글로 전환됩니다.

## 9. 자주 묻는 질문

### Q1. 관리자로 가입했는데 원이 안 보여요.

- 회원가입 시 원 정보를 입력했는지 확인하세요.
- 프로필에서 원 정보(이름, 주소 등)를 입력·수정하면 원이 생성됩니다.

### Q2. 보호자인데 알림장 작성이 안 돼요.

- **연결된 원**이 있어야 합니다.
- 원 검색에서 연결 요청 후, **관리자 승인**이 완료되어야 합니다.
- "연결된 원이 없습니다." 메시지가 나오면 원 검색에서 연결 요청을 진행하세요.

### Q3. 관리자인데 알림장 작성 시 원이 안 보여요.

- 해당 원에 **연결된 반려동물**이 있어야 합니다.
- 원 상세 → 승인 대기에서 보호자의 연결 요청을 **승인**한 후 사용할 수 있습니다.

### Q4. 사진 첨부 제한이 있나요?

- 1~10장까지, 파일당 10MB 이하
- JPEG, PNG, WebP 형식 지원

### Q5. 푸시 알림이 오지 않아요.

- 프로필에서 **알림 받기**를 눌렀는지 확인하세요.
- 브라우저 알림 권한이 허용되어 있는지 확인하세요.
- 모바일에서 사용 시 **홈 화면에 추가** 후 앱으로 실행하면 알림 수신이 더 안정적입니다.

### Q6. 보호자가 알림장 수정·삭제를 할 수 있나요?

- **수정·삭제는 관리자만** 가능합니다.
- 보호자는 알림장 작성·열람·댓글만 가능합니다.
